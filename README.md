# 📚 Book Library Dashboard

A modern and responsive **Book Library Dashboard** built with **React** that allows users to manage their personal book collection. The application supports full CRUD functionality, advanced filtering, favorites, reading status tracking, dark mode, CSV export, and browser persistence using Local Storage.

---

## 🚀 Features

### 📖 Book Management
- ➕ Add new books
- ✏️ Edit existing books
- 🗑️ Delete books with confirmation modal
- 👀 View complete book details in a dedicated modal

### 📚 Library Organization
- 🔍 Search books by title or author
- 🏷️ Filter books by category
- 🔄 Sort books by title, price, or rating
- ❤️ Mark books as favorites
- 📖 Track reading status (To Read, Reading, Completed)

### 📊 Dashboard
- 📚 Total Books
- 📂 Total Categories
- ⭐ Average Rating
- 💲 Average Price
- 📖 Reading Status Statistics

### 🎨 User Experience
- 🌙 Dark Mode with saved preference
- 📱 Responsive dashboard layout
- 🔔 Toast notifications for user actions
- 📭 Friendly empty state message
- 🖼️ Book cover image support

### 💾 Data Management
- 💾 Save library using Local Storage
- 🔄 Automatically restore data on refresh
- 📄 Export library to CSV
